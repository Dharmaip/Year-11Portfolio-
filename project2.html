Python Quiz Game

Overview

This quiz game tests your general knowledge in various topic from biology, history, and more. There are 15 questions you have to answer, but you only have 2 attempts to answer it correctly. If you got it wrong, don't worry, the program will gives you hints to help you answer. Your answer will be stored and calculated for scoring at the end of the game. There are more instruction in the game file comments.

End Users

This game is for everybody and has no age limit. It is designed to be fun and educational for all ages.

Requirement & Specification

  • IDE: This is where I make the website. I use an IDE called Visual Studio Code which is a popular, easy, and extension rich IDE
  • Github: Github is where I put all of my project and host my websites
  • Python: Python is a high-level programming language that I use to create the quiz game. It is known for its readability and simplicity, making it a great choice for beginners.

Relevant Implications

By creating this project, I have improved my skills in Python programming and game development. This project could inspire others to learn Python and create their own quiz games or other types of games. It also demonstrates my ability to create interactive applications that can be used for educational purposes.
